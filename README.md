# AWS assignment

## Overview
This project is a simple frontend and backend application deployed using Docker.
The application connect  to a MySQL database and is deployed on AWS EC2 using Docker containers.

## Tech Stack

- Node.js 
- MySQL
- Docker 
- AWS EC2
- ALB


## Application Details

- Backend runs on 3000
- Database: MySQL (Docker container)

## Local Setup
```bash
git clone <repo-url>
cd aws-assignment
docker compose up --build
